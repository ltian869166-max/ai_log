# 林甜 · 个人作品集

> UI / 视觉设计师 · 个人作品展示网站

本网站为个人作品集展示，不作为商业用途。

## 技术栈

- React 18 + Vite 6
- 纯 CSS 动画（IntersectionObserver）
- 深色系 · 黄绿色点缀

## 本地运行

```bash
pnpm install
pnpm dev
```

浏览器打开 http://localhost:5173

## 部署

项目已配置 GitHub Actions 自动部署到 GitHub Pages。
推送至 `main` 分支后，Actions 会自动构建并部署。
